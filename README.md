# Tab URL Exporter

開いているウインドウ内のすべてのタブのタイトルとURLを取得するChrome拡張機能です。

## 機能

- 現在のウィンドウで開いているすべてのタブのページタイトルとURLを取得
- テンプレート機能で表現形式をカスタマイズ可能（Markdown形式など）
- 区切り文字を編集可能
- ワンクリックでクリップボードにコピー

## インストール方法

1. Chromeブラウザで `chrome://extensions/` を開く
2. 右上の「デベロッパーモード」を有効にする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このフォルダを選択

## 使用方法

1. ブラウザのツールバーにある拡張機能のアイコンをクリック
2. ポップアップが開き、自動的に現在のウィンドウの全タブ情報が取得されます
3. 「copy」ボタンをクリックしてクリップボードにコピー

## カスタマイズ

### テンプレートの変更

`tabUrlExporter.js` の2行目の `DEFAULT_TEMPLATE` 変数を編集することで、出力形式を変更できます。

- `%%title%%`: ページのタイトルに置換されます
- `%%URL%%`: ページのURLに置換されます

例：
- Markdown形式: `[%%title%%](%%URL%%)`
- デフォルト: `[%%title%%](%%URL%% \"%%title%%\")`

### 区切り文字の変更

`tabUrlExporter.js` の3行目の `DEFAULT_DELIMITER` 変数を編集することで、タブ間の区切り文字を変更できます。

例：
- 改行: `'\n'`
- コンマ区切り: `', '`

## 必要な権限

- `activeTab`: アクティブなタブへのアクセス
- `tabs`: すべてのタブ情報の取得
- `clipboardWrite`: クリップボードへの書き込み

## 注意事項

- アイコンファイル（icon16.png, icon19.png, icon48.png, icon128.png）が必要です
- アイコンファイルがない場合は、manifest.jsonからアイコンの設定を削除するか、適切なアイコンファイルを用意してください
